# Triage Labels

工程技能使用 5 个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的状态字符串。

| Role in mattpocock/skills | Status in this repo | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | 维护者需要先评估这个 issue |
| `needs-info` | `needs-info` | 等待 reporter 或人类补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可交给 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 问题明确，但需要人类实现或决策 |
| `wontfix` | `wontfix` | 已确认不会处理 |

当技能提到某个 triage role 时，使用上表右侧的状态字符串。
