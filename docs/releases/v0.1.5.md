# Voily 0.1.5

## 中文

这是一次面向本地 SenseVoice 和上手体验的改进版本。本地识别继续保持模型可下载、可清理的边界，同时减少运行时依赖带来的安装和更新负担。

### 新增和改进

- 本地 SenseVoice 现在使用应用内 MLX Swift 推理，不再要求用户安装 Python 或额外运行时。
- 模型页支持直接下载 SenseVoice Small 的 MLX 模型文件，可在 ModelScope 和 Hugging Face 之间选择下载源。
- 本地模型缓存状态更清楚。Voily 会显示模型是否缺失、正在下载、正在校验或已经可用。
- 支持清除本地 SenseVoice 模型缓存。清除操作只删除大模型权重，不会删除设置或云端 provider 凭证。
- 本地模型下载、校验和识别失败时会显示更明确的错误提示，方便重新下载或定位缺失文件。
- README 和官网入口已更新，下载、快速上手、权限说明和核心能力介绍更容易浏览。

### 兼容性

- 适用于 macOS 14.0 及以上版本。
- 首次使用仍需要授予麦克风权限和辅助功能权限。

## English

This release improves the local SenseVoice workflow and first-run experience. Local recognition keeps the large model files downloadable and removable while reducing installation and update overhead from extra runtime dependencies.

### New and Improved

- Local SenseVoice now runs through the in-app MLX Swift inference path, so users do not need to install Python or a separate runtime.
- The Models page can download the SenseVoice Small MLX model files directly, with ModelScope and Hugging Face available as download sources.
- Local model status is clearer. Voily shows whether the model is missing, downloading, validating, or ready.
- You can clear the local SenseVoice model cache. This only removes the large model weights and does not delete settings or cloud provider credentials.
- Local model download, validation, and recognition failures now show clearer messages to help retry downloads or identify missing files.
- The README and website entry point have been updated so downloads, quick start steps, permissions, and core capabilities are easier to scan.

### Compatibility

- Requires macOS 14.0 or later.
- First-time use still requires Microphone and Accessibility permissions.
