# Voily 0.1.3

这是一次用于验证应用内更新链路的维护版本，重点修复发布产物和更新提示的可见问题。

## 修复和改进

- 修复 GitHub Actions 发布时 DMG 安装窗口可能丢失定制布局的问题。发布流程现在使用 `dmgbuild` 直接写入磁盘镜像展示元数据，不再依赖 Finder/AppleScript GUI 自动化。
- 恢复拖拽安装界面的定制展示：`Voily.app` 位于左侧，`Applications` 快捷方式位于右侧，并使用 `Resources/Release/dmg-background.png` 作为安装窗口背景。
- 改进 Sparkle 应用内更新说明。发布流程会把本文件复制成 DMG 同名 release notes，并嵌入 `appcast.xml`，因此“检查更新...”窗口会显示本次修改内容，而不是极短的默认描述。
- 改进 GitHub Release 说明。发布流程会使用同一份 release notes 创建或更新 GitHub Release，避免 GitHub 页面和 Sparkle 弹窗内容不一致。
- 将应用版本提升到 `0.1.3`，build number 提升到 `3`，用于从 `0.1.2` 验证 Sparkle 下载、安装和重启升级链路。

## 没有变化

- 本版本没有新增 ASR、LLM 润色或触发键交互行为。
- 首次启动仍需要 macOS 麦克风权限和辅助功能权限。
