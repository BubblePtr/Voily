# Voily 0.1.4

## 中文

这是一次面向设置页和权限引导的体验改进版本，重点降低首次配置和权限修复时的摩擦。

### 新增和改进

- 设置窗口重新整理为概览、通用、输入、外观和关于页面，常用选项更容易定位。
- 关于页面现在会显示当前版本信息，并提供手动“检查更新”入口。
- 设置首页新增麦克风和辅助功能权限状态。缺少关键权限时，Voily 会显示更明确的提示，并提供对应的修复入口。
- 返回或重新打开设置窗口时，权限状态会自动刷新，授权后的状态反馈更及时。
- 听写润色会更好地保留已经识别正确的英文内容，减少不必要的翻译或改写。
- 更多设置、权限、模型下载和识别失败提示现在提供英文、日文、简体中文和繁体中文文案。

## English

This release improves the Settings experience and permission guidance, with a focus on making first-time setup and permission recovery easier.

### New and Improved

- The Settings window is now organized into Overview, General, Input, Appearance, and About pages, making common options easier to find.
- The About page now shows the current app version and includes a manual “Check for Updates” action.
- The Settings overview now shows microphone and Accessibility permission status. When a required permission is missing, Voily shows clearer guidance and a direct recovery path.
- Permission status refreshes when you return to or reopen Settings, so granted permissions are reflected more quickly.
- Dictation refinement preserves correctly recognized English more reliably, reducing unnecessary translation or rewriting.
- More settings, permission prompts, model download messages, and recognition failure messages are now available in English, Japanese, Simplified Chinese, and Traditional Chinese.
