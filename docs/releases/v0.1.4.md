# Voily 0.1.4

这是一次面向设置页、安装体验和权限引导的改进版本，重点降低首次配置和权限修复时的摩擦。

## 新增和改进

- 重新整理设置窗口结构，将设置拆分为更清晰的概览、通用、输入、外观和关于页面。
- 在关于页面新增版本信息和手动“检查更新”入口，便于验证 GitHub Release / Sparkle 更新链路。
- 新增麦克风和辅助功能权限状态卡片。应用会在设置页内展示当前权限状态，并提供内置引导入口帮助用户修复权限问题。
- 在缺少关键权限时显示更明确的提示，设置页重新激活时也会刷新权限状态。
- 将本地安装命令拆分为 `make install-dev` 和 `make install-debug`，分别用于 Release 形态验证和 Debug 调试安装。
- 改进文本润色提示词：在听写润色时更好地保留已经识别正确的英文内容。
- 扩充英文、日文、简体中文和繁体中文本地化，覆盖设置页、权限提示、模型/ASR 错误等更多用户可见文案。

## 修复

- 修复部分权限恢复路径只能依赖外部系统设置的问题，现在麦克风与辅助功能权限都有更明确的应用内说明。
- 改进多个 ASR provider 和本地模型流程的错误展示，让常见失败原因更容易定位。

## 没有变化

- 本版本没有改变触发键交互：短按仍为听写切换，长按仍为中英翻译。
- 首次启动仍需要 macOS 麦克风权限和辅助功能权限。
